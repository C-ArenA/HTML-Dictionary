  1. br tag

    This is a self closed tag that lets you go to the next line: <br>

    Example:

    Code:

    
                <p>This text goes before the br tag</p> 
    
                <br> 
    
                <p>This text goes after the br tag</p>

    Result:

    This text goes before the br tag


    This text goes after the br tag



  2. hr tag

    This is like br, but it draws a line, a divider: <hr>

    Example:

    Code:

    
                <p>This text goes before the hr tag</p> 
    
                <hr> 
    
                <p>This text goes after the hr tag</p>

    Result:

    This text goes before the br tag

    This text goes after the br tag
